🚀 Professional Portfolio | Full Stack Developer
A high-performance, responsive portfolio website built for Umuhire Cyusa Alain. This project showcases modern web development practices including custom CSS animations, interactive service cycling, and a mobile-first responsive design.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Flexbox & Grid)

Interactivity: Vanilla JavaScript

Typography: Sora & Inter (Google Fonts)

Icons: FontAwesome 6.0

✨ Key Features
Interactive Service Carousel: A custom-built JavaScript "cycling" system for the What I Do section that allows infinite scrolling through service cards without disrupting the page layout.

Responsive Navigation: A smart dropdown menu that activates at mobile resolutions (768px and below) for seamless multi-device browsing.

Modern UI/UX: Dark mode aesthetic featuring high-contrast gradients, CSS hover transformations, and glassmorphism-inspired project cards.

Scalable Architecture: Clean code structure separating structure (HTML), style (CSS), and logic (JS).
